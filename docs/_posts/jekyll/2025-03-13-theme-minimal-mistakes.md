# jekyll theme minimal mistakes  

 
## 三种使用主题的方法  

### gem theme  
适合自托管（Self-hosting）, 指的是你自己负责运行和托管 Jekyll 生成的网站  

### remote theme  
简单化引入, 下载远程主题的文件时不会覆盖本地文件, 从而依然可以定制化.  

### fork repo  
高度定制化.
